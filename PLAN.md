# qsd Admin 开发计划

## 阶段 0：基础设施

- 搭建 Vue 3 + TypeScript + Vite 管理端
- 搭建 Spring Boot 3 + Spring Security + JWT + MyBatis-Plus + Flyway 后端
- 打通登录、鉴权、菜单、基础布局

## 阶段 1：后台基础能力

- 用户、角色、菜单、权限
- 登录态恢复与路由守卫
- 内容管理基础结构

## 阶段 2：业务模块

- 首页内容管理
- 新闻管理
- 运单管理
- 字典管理

## 阶段 3：会员系统与小程序接入

### 后端已完成

- 新增 `member_user`

- 新增会员注册、登录、资料、运单查询接口
- 新增后台会员管理接口

### 前端已完成

- 新增会员管理菜单与页面
