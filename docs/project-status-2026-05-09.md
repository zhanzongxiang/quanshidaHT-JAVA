# qsd Admin 项目状态

更新时间：2026-05-09

## 当前阶段判断

项目已完成阶段 0-3，当前处于：

- 阶段 4：微信登录与微信支付扩展，已完成大部分后端与后台管理端落地
- 阶段 5：测试补齐、联调、上线准备，尚未完成

当前状态不是“还在搭框架”，而是“主干业务已跑通，支付与微信能力已出第一版，正在收尾和上线化”。

## 已完成

### 1. 后台基础能力

- 管理员登录、JWT 鉴权、菜单、权限、路由守卫已具备
- 后台基础模块已落地：内容管理、新闻管理、运单管理、字典管理
- 前端管理端已具备会员页和支付页

### 2. 会员系统

- 会员数据模型与 Flyway 迁移已存在
- 会员注册、登录、资料查询、资料更新接口已完成
- 会员 JWT 与管理员 JWT 已隔离
- 后台会员管理接口与页面已完成
- 会员与运单绑定、会员查看本人可见运单已完成

### 3. 微信身份接入

- 会员表已包含 `wechatOpenid`、`wechatUnionid`、`wechatBindTime`
- 已提供 `POST /api/member/auth/wechat-login`
- 已提供 `PUT /api/member/profile/wechat`
- 当前实现已支持：
  - 通过微信 `code` 换取会话
  - 按 `openid` 查找会员
  - 首次微信登录自动创建会员或绑定已有会员

### 4. 支付系统 v1

- 已有支付相关数据模型：
  - `pay_order`
  - `pay_transaction`
  - `refund_order`
  - `pay_notify_log`
  - `refund_notify_log`
  - `pay_reconcile_record`
- 后台支付管理接口已完成
- 后台支付管理页面已完成
- 会员侧支付准备与支付记录接口已完成

### 5. 微信支付一期能力

- 小程序支付准备后端链路已具备
- 微信支付回调、退款回调接口已具备
- 商户配置可切换，支付单保留商户快照
- 平台证书支持手动刷新和自动刷新
- 支持下载交易账单并生成本地对账记录
- 同时提供 mock 网关和真实微信网关两套实现

### 6. 小程序前端骨架

- 仓库内已新增 `miniapp/` `uni-app` 小程序骨架
- 已覆盖：
  - 手机号注册与登录
  - 微信登录入口
  - 会员资料查看与编辑
  - 运单列表与详情
  - 支付记录列表
  - 运单详情中的支付准备入口
  - 支付结果页骨架
- 已补充：
  - 未登录访问受限页时的登录后回跳
  - 支付结果页有限轮询和超时收口
  - 支付发起失败时的结果页提示透传
  - 支付记录页直接回看支付结果
  - 资料页微信状态摘要、刷新和重走微信登录入口
  - 首页、运单列表、支付列表统一展示状态文案
  - 小程序内置联调清单页和快捷入口
  - 小程序内置环境诊断页与最近接口记录
  - 联调清单页可本地勾选并统计进度
- 已通过：
  - `npm run type-check`
  - `npm run build:mp-weixin`
- 微信小程序构建目录已确认：
  - `miniapp/dist/build/mp-weixin`
